<h2>Descripción</h2>

<p>Este repositorio es un punto de partida para construir una aplicación con el framework <a href="https://nestjs.com/" target="_blank">Nest</a>.</p>

<h2>Instalación</h2>

<pre>
  <code>$ yarn install</code>
</pre>

<h2>Ejecutar la aplicación</h2>

<pre>
  <code>
    # modo desarrollo
    $ yarn run start

    # modo watch
    $ yarn run start:dev

    # modo producción
    $ yarn run start:prod
  </code>
</pre>

<h2>Pruebas</h2>

<pre>
  <code>
    # pruebas unitarias
    $ yarn run test

    # pruebas e2e
    $ yarn run test:e2e

    # cobertura de pruebas
    $ yarn run test:cov
  </code>
</pre>
